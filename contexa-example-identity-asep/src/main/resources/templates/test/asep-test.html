What is ASEP? Spring's @ControllerAdvice only catches exceptions at the controller level. Spring Security operates in the filter chain BEFORE the servlet, so @ControllerAdvice cannot catch security filter exceptions. ASEP's ASEPFilter catches ALL exceptions in the filter chain and delegates them to @SecurityControllerAdvice handlers.
