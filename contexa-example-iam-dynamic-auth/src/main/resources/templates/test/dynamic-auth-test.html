Public Resource

isAuthenticated()

NORMAL

Normal Resource

#trust.hasActionIn('ALLOW','MONITOR') and hasRole('USER')
AI analysis preferred. ALLOW or MONITOR action accepted.
SENSITIVE

Sensitive Resource

#trust.requiresAnalysisWithAction('ALLOW','MONITOR') and hasRole('USER')
AI analysis must complete before access. Stricter than normal.
ADMIN

Admin Resource

hasRole('ADMIN') and #trust.requiresAnalysisWithAction('ALLOW')
ADMIN role + AI must return ALLOW. Most restrictive.
